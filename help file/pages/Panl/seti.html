System.ArgumentExceptionThrown when image.Height > 480 or image.Width > 640
System.IndexOutOfRangeExceptionThrown when index is outside the bounds of the array

Remarks

New image must be 640x480px or smaller. PixelFormat is not strict, image will be converted to 8bppIndexed and match the original Palette as best as possible.

If index is not specified, defaults to zero and the first image will be replaced.
