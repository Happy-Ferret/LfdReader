NameDescription
System.ArgumentExceptionThrown when image.Height > 480, image.Width > 640, or image contains a line of exactly 256 or 512 consecutive pixels of a given color

Remarks

New image must be 640x480px or smaller. Image does not have to be of a mask type, the original image can be used as long as the transparent regions are a single color not used elsewhere in the image. PixelFormat is not strict, image will be converted to 1bppIndexed. If transparentColor is omitted, black (#000000) will be used.

The 256/512px restriction is in place due to how the rows are encoded. Cannot have a 256/512 line in the middle of an image, but it can end in one.
