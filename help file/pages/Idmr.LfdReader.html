LfdReader Namespace

The LfdReader namespace contains all of the functions required to read/write LFD Resource files

Classes

NameDescription
AnimReads LFD files and interprets ANIM animation resources
BlasReads LFD files and interprets BLAS and VOIC audio resources
DeltReads LFD files and interprets DELT image resources
FilmReads LFD files and interprets FILM layout resources
FontReads LFD files and interprets FONT resources
MaskReads LFD files and interprets MASK transparency resources
PanlReads LFD files and interprets PANL cockpit image resources
PlttReads LFD files and interprets PLTT color palette resources
ResourceBase class for LFD resources, also provides ability to retrieve raw data from unknown resources
RmapReads LFD files and interprets RMAP header resources
TextReads LFD files and interprets TEXT resources
