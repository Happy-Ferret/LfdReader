Font.BaseLine

Namespace: Idmr.LfdReader
Assembly: Idmr.LfdReader.dll
Syntax: public short BaseLine { get; set; }

Gets or sets the zero-indexed ScanLine that is used as the "bottom" of the font.

Remarks

Adjusting this value determines "dip" for characters such as g, j, p, etc and how much white space is required under normal characters.
