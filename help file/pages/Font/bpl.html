Font.BitsPerScanLine

Namespace: Idmr.LfdReader
Assembly: Idmr.LfdReader.dll
Syntax: public short BitsPerScanLine { get; set; }

Gets or sets the length of values required per scanline

Exceptions

NameDescription
System.ExceptionThrown if value is negative, or not a multiple of eight.

Remarks

When setting, value must be a multiple of eight. Effectively determines number of bytes to be read per character. Higher values allow for wider characters.
