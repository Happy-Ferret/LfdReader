Delt.ErrorImage

Namespace: Idmr.LfdReader
Assembly: Idmr.LfdReader.dll
Syntax: public static System.Drawing.Bitmap ErrorImage { get; }

Returns a place-holder image.

Remarks

This is a 120x120px image with the phrase "An error has occurred and the image can't be processed" on it. Used when internal exceptions are caught. Forced exceptions occur during processing when the pixel being processed is outside the defined size of the image.
