Remarks

deltPalette should be 256 colors, although typically only a narrow band of the entries will be used. When using the palettes array, will effectively create deltPalette by combining the Pltts in array order. If there are any overlaps, the last values will be used.

animX and animY are used to create the full-sized image defined in an Anim object. The Delt image will be located within the image so that the animation will display correctly.
