Exceptions

NameDescription
System.ArgumentExceptionThrown when image is not PixelFormat.Format8bppIndexed (EncodeImage(Bitmap) overload only)

Remarks

The bulk of the Write operations for Delt and Anim, this takes image and converts it to the prepared raw byte data to be written to file.
If palette is used, image is converted via Idmr.Common.Graphics.ConvertTo8bpp( ) before encoding.
