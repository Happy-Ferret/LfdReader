System.Drawing.Bitmap ImageThe 256-color image

Remarks

Within the Anim class, no direct access is given to the Frame array, this keeps validation external to the struct.

Left and Top should be set so that the edges of Image do not fall outside the extent of the parent animation. Image should be 256-colors and no larger than the parent animation in either dimension.
