Text.NumStrings

Namespace: Idmr.LfdReader
Assembly: Idmr.LfdReader.dll
Syntax: public short NumStrings { get; set; }

Gets or sets the number of strings in the resource

Remarks

Text.Strings will automatically adjust size as required. If new value is smaller, any extra strings beyond that index will be lost. If new value is larger, new empty strings will be null.
