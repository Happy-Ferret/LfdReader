Pltt.Palette

Namespace: Idmr.LfdReader
Assembly: Idmr.LfdReader.dll
Syntax: public System.Drawing.Imaging.ColorPalette Palette { get; }

Gets a 256-color ColorPalette of the resource entries.

Remarks

Palette is initialized as a 256-color indexed palette. The Pltt.Entries array is then laid over the default. Palette is best used for large palettes typically seen with background images. It is not suggested to be used for smaller palettes that have to be combined with others to create the final color set.
