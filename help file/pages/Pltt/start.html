Pltt.StartIndex

Namespace: Idmr.LfdReader
Assembly: Idmr.LfdReader.dll
Syntax: public byte StartIndex { get; }

Gets the index of the first entry i nthe resource

Remarks

This is first entry, Pltt does not have a length value. So a StartIndex of 10 and a EndIndex of 20 would be a total of 11 colors.
