paletteThe palette being used
redThe R component, 0-255

Returns

Index of the given color from palette

Remarks

Any duplicate colors will return as the first possible index. TIE tends to use #000000 as transparency a lot, so if the image uses black as both a normal color and transparency, all black will be returned as transparency (index 00).
